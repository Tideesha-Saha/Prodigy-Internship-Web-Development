Responsive Landing Page

Features

This landing page demonstrates a clean and responsive layout built using HTML, CSS, and JavaScript. The key features include:

  • Fixed Navigation Bar: Stays visible during scrolling and dynamically changes background color on scroll.
  • Interactive Hover Effects: Menu items scale and highlight on hover for a modern UI experience.
  • Responsive Design: Adjusts seamlessly across all screen sizes â€” desktop, tablet, and mobile.
  • Hero Section: Includes a gradient background, custom shapes, and a flexible layout with image and heading.
  • Flexbox Projects Section: Uses Flexbox for layout of project cards, ensuring consistent spacing and alignment.
  • Clean Typography and Styling: Consistent fonts, spacing, and color palette for visual appeal and readability.

About This Task

This project was developed as part of the Web Development Internship at Prodigy Infotech. The primary goal was to design and build a responsive landing page that demonstrates key frontend development skills.

Through this task, I gained hands-on experience in structuring web pages using HTML, styling them with modern CSS techniques such as Flexbox and gradients, and adding interactivity with JavaScript. The fixed navigation bar with scroll-based styling, hover animations, and fully responsive layout are highlights of this task.

This project reflects the foundational skills essential for building engaging and user-friendly web interfaces, and sets the stage for more advanced frontend projects.
